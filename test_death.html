JSRogue - Death System Test

Death Test Controls

Use these buttons to test the death system:

Test Instructions:

  • Kill Player with Damage: Directly reduces HP to 0
  • Kill Player with Poison: Adds poison status effect that kills
  • Start Battle: Initiates a battle where you can lose
  • Check Player Status: Shows current HP and status effects
  • Heal Player: Restores player to full health

Expected Behavior:

  • When player dies, a red death modal should appear
  • Modal should show final statistics (level, XP, gold, enemies defeated)
  • Three restart options should be available:
    • Restart Same Level: Keeps current level, new map
    • Restart New Level: Starts at level 1, new map
    • View Final Stats: Shows detailed statistics
  • Player movement should be disabled when death modal is open
  • Escape key should not close the death modal
